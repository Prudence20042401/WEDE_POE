# Kopano Construction Skills Centre — Website
## Project overview
This repository contains the website for Kopano Construction Skills Centre. The site presents courses, a gallery, enquiry and contact forms, and contact location.

## How to run locally
1. Open the folder in VS Code (recommended).  
2. Install Live Server extension.  
3. Right-click `index.html` → Open with Live Server.  
4. Test contact and enquiry forms locally (contact uses mailto which opens an email client).
