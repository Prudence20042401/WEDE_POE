# README for Kopano Construction Skills Centre

## Project overview
A website to present Kopano Construction Skills Centre, its courses, and contact/enquiry options. This repository contains the website files (HTML, CSS, JS, images) and supporting files for Part 3: Enhancing Functionality & SEO.

## Part 3 — Summary
This repo now includes:
- JavaScript features: mobile menu, accordion, gallery lightbox, search filter, form validation, enquiry response.
- SEO: meta tags on pages, robots.txt, sitemap.xml, OpenGraph tags.
- Accessibility improvements: `alt` text, `aria-label`, semantic tags, keyboard support.
- `part3_improvements.md` - the improvement document showing what was changed and why.

## How to run locally
1. Install VS Code (recommended).  
2. Open this folder in VS Code.  
3. Install Live Server extension (Ritwick Dey).  
4. Right-click `index.html` → **Open with Live Server**.  
5. Check `contact.html` and `enquiry.html` for forms and validation.
